This project is made using open cv , no concept of deep learning used

Steps:

1. Read the image and convert it into grayscale
2. Blur the grascale image using Gaussian Blur
3. Use canny edge method to detect the edges
4. find coutours
5. Get the location of License Plate using contours
6. Make mask and then apply it on original image
7. Use tesserect to read string from images 

Sample images are proveded for Testing
